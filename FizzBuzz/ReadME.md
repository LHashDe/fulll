FizzBuzz

This is a simple FizzBuzz implementation in JavaScript made by Hai-Dang LE

How to Run

You have two ways to run the program:

1. Using a Web Browser

Open the index.html file in your browser.

Open the browser console to see the output.

2. Using Node.js

Run the program with the default value of 100:

node main.js

Or run with a custom number N:

node main.js <number>

If no number is provided, the program defaults to N = 100.
